# Spotify-clone
A frontend clone of Spotify built to practice responsive web design and modern UI development. The project replicates Spotify’s layout, navigation, and music player interface using HTML, CSS, and JavaScript. It focuses on creating a clean, user-friendly design and demonstrates skills in styling, layout structuring, and interactive elements.
Features-
- Responsive layout for desktop and mobile
- Navigation bar with playlists
- Music player interface with play/pause controls
Tech Stack- HTML, CSS (Flexbox/Grid), JavaScript (DOM manipulation)
What I Learned
- learned how to structure responsive layouts, replicate real-world UI, and use JavaScript for interactivity.
